Name

lm - ls for music

Synopsys

lm [ --author | --genre | --album | --track ]… [ --header ] file…

Description

lm is a little program who read multimedia metainformations, such as genre, title, track, album and also artist for a music file and prints it in human-readable way. It’s seen as an a music dedicated ls(1) varient.

The output list always diplay informations in the following order: Title name, Track number, Author name, Album, Genre.

  1. test

Sort options

It is possible to combine the multiple existing sort options but the order you mention it have influence on the sorting. Then if you use --genre before --author, you will have the same artist track in differente genre group, if this artist have two tracks of different genre.

--album
Sort the given files by album name.
--author
Sort the given files by author.
--genre
Sort the given files by genre.
--track
Sort the given files by track number.

Other options

--header
Display a header on the top of the list remembring the colomn information type.
